Ferramentas utilizadas: LESS, HTML, React JS, Node JS

-->
Um aplicativo de rastreamento de condicionamento físico que ajuda os usuários a gerenciar seus treinos. Os usuários podem adicionar esquemas de exercícios ao aplicativo e manter registros de todas as séries, repetições e pesos do treinamento.
Os usuários precisam se autenticar para acessar seus perfis de treinamento. As informações do usuário são salvas no banco de dados para acompanhar todos os seus treinamentos e planos de levantamento.
Pré-processador LESS utilizado para estilizar o aplicativo. Seguiu os princípios de pré-processamento e aproveitou recursos como mixins, variáveis, aninhamento e importações.
